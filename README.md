# Twibbon Merdeka Ride & Evoria Rally 2026

Paket ini sudah siap untuk GitHub Pages.

## PENTING - struktur repository

Setelah ZIP di-extract, upload SEMUA isi folder ini ke halaman utama repository:

```text
assets/
index.html
style.css
script.js
.nojekyll
README.md
```

`index.html` harus terlihat langsung di halaman utama repository, sejajar dengan folder `assets`.

## Cara upload ke GitHub

1. Extract `twibbonhdci-ready-upload.zip`.
2. Buka folder hasil extract.
3. Pilih seluruh isi folder: `assets`, `index.html`, `style.css`, `script.js`, `.nojekyll`, dan `README.md`.
4. Di repository GitHub pilih **Add file → Upload files**.
5. Drag seluruh isi tersebut ke GitHub.
6. Klik **Commit changes**.
7. Buka **Settings → Pages**.
8. Pilih **Deploy from a branch**, branch **main**, folder **/(root)**.
9. Klik **Save**.
